# 🎬 Movie Recommendation System

[View my live portfolio here!](https://phantomdeluxe-dev.github.io/movie-recommendation-system/)

A simple and interactive Movie Recommendation System built using HTML, CSS, and JavaScript. The application allows users to search and filter movies based on language, genre, and ratings, helping them discover movies that match their interests.

## ✨ Features
- Search movies by title
- Filter movies by language and genre
- View movie ratings and details
- Get movie recommendations
- Responsive and user-friendly interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
```
Movie-Recommendation-System/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
